# Uber Clone Backend

A Spring Boot backend system that simulates core Uber functionality including ride booking, driver assignment, ride lifecycle management, and fare calculation.

---

## Features

- User Registration (Rider / Driver)
- Driver Availability Management
- Ride Request & Auto Driver Matching
- Ride Lifecycle:
  - Request
  - Accept
  - Start
  - Complete
- Automatic Fare Calculation
- Secure API Responses using DTO (Passwords hidden)
- Ride Details API

---

## Tech Stack

- Java
- Spring Boot
- Spring Data JPA / Hibernate
- MySQL
- Maven

---

## APIs

### Auth
- Register User

### Driver
- Register Driver
- Set Availability

### Ride
- Request Ride
- Accept Ride
- Start Ride
- Complete Ride
- Get Ride Details
